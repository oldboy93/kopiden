# Kopiden by UAY Coffee Ordering App

## Description

Web application for ordering coffee and snacks for Kopiden by UAY.

The application allows customers to browse menu, add items to cart,
make online payment, and show proof of payment to cashier.

## Users

Customer
Admin

## Features

Customer

- login with email
- browse menu
- filter by category
- add items to cart
- edit cart
- checkout
- online payment via Midtrans
- view order status
- show payment proof

Admin

- login
- create menu
- edit menu
- delete menu
- upload menu image
- manage orders
- update order status

## Platform

Responsive web application
mobile first
